
TC-01 — E2E User flow 

Preconditions: User is on the catalog page
Step	Action	                     Expected Result
1	    - Search item	               - Product list appears matching the search query
2	    - Apply filters              - Product list updates according to filters; irrelevant items hidden
3	    - Sort results	             - Products sorted as selected (e.g., Price Low → High)
4	    - Open product card	         - Product detail page opens showing name, price, description, Add to Cart button
5	    - Add to cart	               - Product added to cart; confirmation message displayed
6	    - Confirm order              - Checkout page opens or order summary confirms addition of product
