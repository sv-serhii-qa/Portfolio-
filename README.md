# Portfolio-
My testing project as QA Manual

   Для  дослідження та демонстрації підходу до тестування, мною був взятий e-commerce/online store [https://volti.ua/uk](url)  
   
Це досить популярна мережа магазинів у моєму регіоні з власним сайтом, і вибрав я його адже це популярний серед проектів напрям у e-commerce, досвід роботи з яким буде вкрай корисним.  

  В ході дослідження для роботи брав за основу основний user flow, для чого власне і створювався даний проект, а також сторонні перевірки які мені були доступні, не завдаючи зайвої шкоди продукту, створюючи "пусті" замовлення.  
  У звіті про тестування будуть додані результати, знайдені дефекти та виявлені ризики, якщо такі матимуть місце.
  
  Для детального визначення основних блоків та структури сайту, я створив Mind map де відтворив сценарій поведінки користувача, що дасть змогу краще проаналізувати повноту тестового покриття.
