# Test object - electronics online store "Volti"


   Для  дослідження та демонстрації підходу до тестування, мною був взятий e-commerce/online store [https://volti.ua/uk](url)   
   
Це досить популярна мережа магазинів у моєму регіоні з власним сайтом, і вибрав я його адже це популярний серед проектів напрям у e-commerce, досвід роботи з яким буде вкрай корисним.  

  В ході дослідження для роботи брав за основу основний user flow, для чого власне і створювався даний проект. Сайт не передбачає реєстрації та авторизації користувача, отже основана роль користувача тут "guest". Також проводились сторонні перевірки які були доступні, не завдаючи зайвої шкоди продукту, генеруючи "пусті" замовлення.  
  У звіті про тестування будуть додані результати, знайдені дефекти та виявлені ризики, якщо такі матимуть місце.
  
  Для детального визначення основних блоків та структури сайту я створив Mind map, де відтворив основний сценарій поведінки користувача що дасть змогу краще проаналізувати повноту тестового покриття, та створити тест кейси на її основі.
